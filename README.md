### 작업 → Commit → Push 순서

1. 코드 수정 or 파일추가

2. GitHub Desktop에서 변경 사항 확인

3. Summary에 작업 내용 요약 작성

4. Commit to [branch 이름] (없으면 적당히 branch 생성)

5. Push origin

6. Push 후 GitHub 웹 페이지에서 Pull Request(PR) 생성 -> 관리자가 리뷰 승인 후 병합



### 사전설정

#### GitHub Desktop 설치 및 초기세팅

1. 아래 공식 링크에서 GitHub Desktop을 다운로드합니다.
👉 https://desktop.github.com/

2. 설치 후 실행합니다.

3. 실행 시 GitHub 로그인을 요청하면 본인의 GitHub 계정으로 로그인합니다

4. GitHub Desktop 실행

5. 상단 메뉴에서 File → Clone repository…

6. URL 탭 선택, 저장소 주소 입력
